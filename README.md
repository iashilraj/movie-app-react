<h1 align="center">Movie App</h1>

<p align="center">A simple React app for searching and displaying movie information using the OMDB API.</p>

<h2 align="center">Features</h2>

- Search for movies
- View movie details
- Responsive design

<h2 align="center">Demo</h2>

<p align="center">Visit the <a href="https://iashilraj.github.io/movie-app-react/">Movie App Demo</a> to explore the application.</p>

<h2 align="center">Installation</h2>

<ol>
  <li>
    Clone the repository:
    <pre><code>git clone https://github.com/iashilraj/movie-app-react.git</code></pre>
  </li>

  <li>
    Change into the project directory:
    <pre><code>cd movie-app-react</code></pre>
  </li>

  <li>
    Install dependencies:
    <pre><code>npm install</code></pre>
  </li>

  <li>
    Start the application:
    <pre><code>npm start</code></pre>
  </li>

  <li>
    Open your browser and navigate to <a href="http://localhost:3000">http://localhost:3000</a> to view the app.
  </li>
</ol>

<h2 align="center">Usage</h2>

- Enter a movie title in the search input field.
- Click on the search icon or press Enter to search for movies.
- View the list of movies matching the search query.
- Click on a movie card to view more details.

<h2 align="center">API Key</h2>

The app uses the OMDB API for movie data. Make sure to replace the `API_KEY` in the `API_URL` variable in `App.js` with your own OMDB API key.

<code>const API_URL = "https://www.omdbapi.com/?apikey=YOUR_API_KEY";</code>

<h2 align="center">Technologies Used</h2>
- React
- OMDB API
- GitHub Pages (for deployment)
